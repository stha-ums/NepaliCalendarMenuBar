## [0.1.0] - 2025-11-16

## Initial Release

This is the first release of Nepali Date Mac Menu Bar.

### Features
- Dual calendar system (Bikram Sambat and Gregorian)
- Multi-language support (English and Nepali)
- Multiple view modes (Monthly and Schedule)
- macOS Calendar integration
- Customizable date formats
- Privacy-focused local processing

## [0.0.1] - 2025-01-17

### Added
- Initial release of Nepali Date Mac Menu Bar
- Dual calendar system (Bikram Sambat and Gregorian)
- Multi-language support (English and Nepali)
- Multiple view modes (Monthly grid and Schedule timeline)
- Customizable date formats (Short, Long, Full)
- macOS Calendar integration with event display
- System integration (Menu bar app with launch on login)
- Privacy-focused local date conversions
- Support for BS dates from 2000 to 2100 (1943 AD - 2043 AD)
- Color-coded calendar events
- Light and dark mode support
- Onboarding experience for first-time users
